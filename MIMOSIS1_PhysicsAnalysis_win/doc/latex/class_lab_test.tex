\doxysubsubsection{\texorpdfstring{plot\_S\_curves()}{plot\_S\_curves()}}
{\footnotesize\ttfamily void Lab\+Test\+::plot\+\_\+\+S\+\_\+curves (\begin{DoxyParamCaption}{ }\end{DoxyParamCaption})}



Creates T\+Graph for each pixel, fits E\+RF function, extracts parameters. 

For each pixel the T\+Graph of number of entries vs scanned parameter value is created. For each T\+Graph the error function\+:

\begin{eqnarray*} f(x) = \frac{1}{2} \cdot N \cdot TMath::Erf(\frac{x - \mu}{\sqrt{2}\cdot \sigma}) \end{eqnarray*}

is fitted and noise ( $ \sigma $) and mean ( $ \mu $) are passed to proper output histograms.

There are two conditions that fit is not proper\+: 0.\+1 $<$ noise $\vert$$\vert$ noise $>$ 10 0 $<$ mean $\vert$$\vert$ mean $>$ \#\+\_\+max\+\_\+param\+\_\+val
